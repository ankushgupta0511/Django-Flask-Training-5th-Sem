# flask with mysql
```
1) download the xampp server.
2) start the apache and mysql server.
error :- if mysql server is not start then uninstall the xampp and delete xampp folder from c derive and again install xampp server


3) create app.py
4) create requirements.txt write all liberary in requirements.txt then run requirements.txt 
error :- if error version related  then directly run pip install liberary


5) open web browser 'http://localhost/phpmyadmin/'
6) click database and create database on browser.

7) any database connectivity use this kind of liberary  'SQLAlchemy'
```
